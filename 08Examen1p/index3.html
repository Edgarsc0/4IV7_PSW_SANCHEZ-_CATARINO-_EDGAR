EXAMEN 1er PARCIAL

Requerimientos

Se deberá de crear una tabla dinámica con HTML + CSS3 + JavaScript para el cálculo de interés compuesto para la compra de un automóvil considerando los siguientes elementos: Valor del Automóvil (Deberá de ingresarlo el usuario el rango de valores es de 100,000.00 a 1,000,000.00) Cuota inicial (Deberá de ingresarlo el usuario y su valor mínimo es de 10% del valor del automóvil) Periodo (Es el número de meses con que se podra liquidar el valor del automóvil las opciones son 3, 6, 9, 12, 18 meses) Interés 3,94% (Se deberá de realizar el cálculo correspondiente del valor del interés). Cuota (Es el valor del pago fijo que se deberá de efectuar de forma mensual). Considere la siguiente formula: S =P(1 + I)n S es el valor futuro del crédito, es decir, el valor inicial del crédito más los intereses capitalizados. P es el valor presente del crédito, es decir, el valor inicial de crédito. I es la tasa de interés expresada en decimales (5% = 0,05 que resulta de 5/100). N es el periodo o número de meses de plazo del crédito. https://www.gerencie.com/interes-compuesto.html Para realizar la evaluación del examen: Deberá de estar programado em Javascript y poder elegir con los botones para realizar el calculo del interés compuesto Deberá de tener un estilo único siguiendo las reglas de CSS y JavaScript Deberá de contener los datos del alumno en el footer Deberá de tener el enlace desde su index del repositorio programado. Deberá de tener una css con 3 colores base. Se debe validar la entrada del formulario acorde a las condiciones y lógica necesaria. El usuario podra modificar los valores del cálculo del interés compuesto según: elegir el periodo, cambiar los montos de la cuota inicial
